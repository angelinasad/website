Костюмы для маленьких собак

Комбинезон для маленьких собак

Подходит для зимних прогулок. Выполнен из нескольких слоев ткани, сохраняет температуру тела собаки. Подробнее 2500 руб.

Свитер для маленьких собак

Подходит для осенних/весенних прогулок или для дома. Выполнен из натуральных материалов, поэтому его можно носить каждый день! Подробнее 999 руб.

Дождевик для собак малых пород

Подходит для дождливых прогулок. Состоит из водонипроницаемой ткани, его можно носить как летом, так и зимой. Подробнее 690 руб.
